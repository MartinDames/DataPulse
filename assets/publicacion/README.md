# Screenshots para publicación

Capturas de DataPulse listas para LinkedIn e Instagram.

**Regenerar** (con dev server en `localhost:3000`):

```powershell
cd "...\demos\data-health"
npm run dev
# En otra terminal, desde PUBLICAR.md o scripts locales con Playwright
```

## Archivos

| Archivo | Uso |
|---------|-----|
| `01-landing-hero.png` | Post lanzamiento · slide 1 Instagram |
| `02-overview-linkedin.png` | **LinkedIn post principal** (viewport, score crítico) |
| `02-overview.png` | Overview página completa (referencia / web) |
| `03-alertas-linkedin.png` | **LinkedIn / IG** — lista de tickets urgentes |
| `03-alertas.png` | Alertas página completa (carrusel detalle) |
| `04-informe.png` | Carrusel Instagram slide informe |
| `05-landing-solucion.png` | Sección solución (3 tarjetas) |

## Recomendación rápida

- **LinkedIn lanzamiento:** `02-overview-linkedin.png`
- **Instagram carrusel slides 4–5:** `02-overview-linkedin.png` + `03-alertas-linkedin.png`
- **WhatsApp:** cualquiera de los `*-linkedin.png` (pesan menos, se ven bien en móvil)
