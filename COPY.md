## Problema (3 dolores)

1. **Nadie mira la base hasta que explota** — el error aparece en facturación, stock o reportes, no en IT.
2. **Excel y sistema no cierran** — dos versiones de la verdad; nadie sabe cuál confiar.
3. **Backups y jobs en piloto automático** — asumís que funcionan hasta que no.

---

## Solución (qué hace DataPulse)

- **Score de salud (0–100)** — Indicador 0-100 con semaforo para una rapida lectura, no solo del equipo técnico.
- **Alertas accionables** — severidad, impacto y recomendación en lenguaje claro.
- **Informe ejecutivo** — Resumen de una pagina, listo para presentar.

---

## Checks que detectamos (demo)

| Check | Ejemplo |
|-------|---------|
| Backups | Último backup > 24 h sin verificar |
| Duplicados | Registros repetidos en tablas críticas |
| Performance | Consultas lentas recurrentes |
| Crecimiento | Tablas con crecimiento anormal |
| Reconciliación | Excel vs ERP no coincide |
| Infra | Espacio en disco / conexiones al límite |
| Jobs | Cron o ETL fallidos sin aviso |

---

## Para quién es

- PyME con PostgreSQL, SQL Server o datos en la nube
- Dependencia de Excel + sistema que no cierran
- Sin DBA interno o con IT saturado
- Dueño o gerente que quiere visibilidad sin entrar al SQL

## Para quién NO es

- Startups con Datadog + equipo SRE
- Empresas que solo buscan una web o redes sociales
- Quien no tiene al menos una fuente de datos digital
